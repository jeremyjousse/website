# Technology Stack

## Core Framework
- **Svelte 5 & SvelteKit 2:** Leverages the latest Svelte features (Runes) and SvelteKit's robust routing and static site generation capabilities.
- **TypeScript:** Ensures type safety and improves developer experience across the codebase.

## Styling and UI
- **Tailwind CSS 4:** Modern, utility-first CSS framework for rapid and maintainable styling.
- **PostCSS:** Used for processing CSS and supporting Tailwind's advanced features.

## Content Management
- **mdsvex:** Allows writing blog posts in Markdown while retaining the ability to use Svelte components within them.
- **Markdown (CommonMark/GFM):** The standard for content authoring in the project.

## Quality Assurance
- **Vitest:** Fast, Vite-native unit and component testing framework.
- **Playwright:** Reliable end-to-end testing for critical user paths.
- **ESLint & Prettier:** Ensures code consistency and enforces best practices.

## Build and Deployment
- **Vite:** Next-generation frontend tooling for fast development and optimized builds.
- **SvelteKit Static Adapter:** Configured to generate a fully static site for deployment to platforms like GitHub Pages.
